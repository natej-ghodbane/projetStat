1 Introduction

The objective of this report is to identify dietary patterns using principal component analysis (PCA) and to explore their associations with blood lipid profiles, based on the logic of the article Association between dietary patterns and blood lipid profiles among Chinese women. We apply the same type of methodology to our own dataset composed of dietary intake (c12diet.csv) and blood biomarkers (biomarker.csv).

2.2 Dietary variables and PCA

We use the following dietary variables, which summarize 3-day intake:

  • d3kcal: total energy (kcal)
  • d3carbo: carbohydrate intake
  • d3fat: fat intake
  • d3protn: protein intake
diet_vars <- data[, c("d3kcal", "d3carbo", "d3fat", "d3protn")]

# Standardize before PCA
diet_scaled <- scale(diet_vars)

We conduct PCA with varimax rotation and extract 3 components, as in the reference study.

library(psych)

pca <- psych::principal(diet_scaled, nfactors = 3, rotate = "varimax")
print(pca, digits = 3)

The factor loadings correspond to three main patterns:

  • Pattern 1: carbohydrate-rich pattern (high loading on carbohydrates)
  • Pattern 2: fat-rich pattern (high loading on fat)
  • Pattern 3: protein-rich pattern (high loading on protein)

2.5 Pattern scores and quartiles

We compute component scores for each participant and create quartiles (Q1Q4) for each dietary pattern, following the reference article.

scores <- as.data.frame(pca$scores)
colnames(scores) <- c("Pattern1", "Pattern2", "Pattern3")  # 1: carbs, 2: fat, 3: protein

data <- cbind(data, scores)

data$Pattern1_Q <- cut(data$Pattern1,
                       quantile(data$Pattern1, probs = seq(0,1,0.25), na.rm = TRUE),
                       include.lowest = TRUE)
data$Pattern2_Q <- cut(data$Pattern2,
                       quantile(data$Pattern2, probs = seq(0,1,0.25), na.rm = TRUE),
                       include.lowest = TRUE)
data$Pattern3_Q <- cut(data$Pattern3,
                       quantile(data$Pattern3, probs = seq(0,1,0.25), na.rm = TRUE),
                       include.lowest = TRUE)

4 Discussion

Using PCA on energy and macronutrient intake variables, we identified three dietary patterns: a carbohydrate-rich, a fat-rich, and a protein-rich pattern. These patterns are conceptually similar to the dietary profiles described in the reference paper, although our variables are nutrients rather than detailed food groups.

In regression models, we observed that:

  • The carbohydrate-rich pattern tends to be associated with lower LDL-C and total cholesterol.
  • The fat-rich and protein-rich patterns tend to be associated with higher triglycerides and total cholesterol.
  • No strong association was observed between the dietary patterns and HDL-C in this dataset.

These findings partially resemble those reported in the original article, where more Western or high-fat patterns were associated with adverse lipid profiles. Differences between our results and the published study can be explained by differences in population, variables (nutrient-based vs food-based patterns), and the lack of adjustment for confounders such as age, BMI, smoking, and physical activity in our models.

Overall, this analysis successfully reproduces the statistical logic of the original study (PCA, factor scores, quartile creation, and regression models) on our own dataset.
